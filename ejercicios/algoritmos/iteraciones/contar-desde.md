Contar Desde
============

Escribir un programa que nos pida un numero, y luego imprima numeros de uno en uno,
desde el numero ingresado inclusive, hasta el numero `0`.

```java
public class ContarDesde {
  public static void main(String[] args) {
    // (tu algoritmo aca)
  }
}
```

#### Ejemplo:

```
Ingrese un numero:
> 4
4
3
2
1
0
```
