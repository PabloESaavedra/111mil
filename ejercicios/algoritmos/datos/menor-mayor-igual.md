Menor, Mayor o Igual
====================

Escribir un programa que nos pida el ingreso de 2 numeros: `A` y `B`, y luego
los imprima segun corresponda:

- A es menor que B
- A es mayor que B
- A y B son iguales

```java
public class MenorMayorIgual {
  public static void main(String[] args) {
    // (tu algoritmo aca)
  }
}
```

#### Ejemplo:

```
Ingrese valor A:
> 30
Ingrese valor B:
> 15
A es mayor que B
```
