Futbol
======

![Futbol](futbol.jpg)

- Los jugadores solo podran jugar en un unico equipo. 

- Los equipos deberan presentar el listado de jugadores al inciar el campeonato. No se aceptaran cambios.

- Se podran jugar partidos de practica, presentando dos equipos: local y visitante.


# Reglas:


**Posiciones**:

|  Pos.  | Tipo      |
| ------ | --------- |
|   1    | Arquero   |
| 2 -  4 | Defensor  |
| 5 -  8 | Central   |
| 9 - 11 | Delantero |

---

**Puntos**:

| Resultado | Puntos |
| --------- | ------ |
| Ganador   |  +3    |
| Empate    |  +1    |
| Perdedor  |   0    |


---

**Tabla**:

Tabla de ejemplo. Dado los siguientes partidos:

    A-B : 1-0
    A-D : 2-2
    B-D : 4-1

__Donde__:

    PJ: Partido Jugado
    PG: Partido Ganado
    PE: Partido Empatado
    PP: Partido Perdido
    DG: Diferencia Goles
    Ps: Puntos

__Posiciones__:

| Equipo | PJ | PG | PE | PP | DG | Ps |
| ------ | -- | -- | -- | -- | -- | -- |
| Eq A   |  2 |  1 |  1 |  0 | +1 |  4 |
| Eq B   |  2 |  1 |  0 |  1 | +3 |  3 |
| Eq D   |  2 |  0 |  1 |  1 | -3 |  1 |
